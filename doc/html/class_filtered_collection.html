Description détaillée

template<typename T>
class FilteredCollection< T >

Collection filtrée

Paramètres du template
TType des éléments de la Collection

Collection qui filtre les éléments d'une autre Collection sur laquelle elle possède une référence. Elle utilise un prédicat défini en tant que std::function et passé à la construction.
